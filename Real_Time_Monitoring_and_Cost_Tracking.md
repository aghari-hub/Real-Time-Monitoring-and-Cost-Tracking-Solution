# Real-Time Monitoring and Cost Tracking on AWS (CloudWatch + SNS)

## Overview
This project sets up real-time monitoring and cost tracking on AWS using Amazon CloudWatch, SNS, and Billing Alerts.

## Architecture
![Monitoring Architecture](sandbox:/mnt/data/A_flowchart_in_a_digital_illustration_depicts_an_A.png)

## Prerequisites
- AWS account
- EC2 or other resources to monitor
- Valid email for notifications

## Steps
1. Create SNS Topic
2. Subscribe email
3. Create CloudWatch Alarms
4. Create Budget Alert
5. Test alert triggers

## Key Learnings
- Using SNS for alerts
- Monitoring metrics with CloudWatch
- Setting AWS Budgets for cost control
